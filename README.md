easyminer
=========

- качаем
http://download.qt-project.org/official_releases/qt/5.2/5.2.1/qt-opensource-windows-x86-mingw48_opengl-5.2.1.exe

- устанавливаем

- щёлкаем по easyminer.pro
- в открывшемся QtCreator щёлкаем по зелёной стрелочке (слева внизу)
